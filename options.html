Screen capture settings

Enabling screen capturing will add more memory usage to chrome and can cause significant CPU usage. Enabling high quality screen captures will increase the memory usage even further.
Also, the screen capturing feature is quite experimental. If you notice strange behaviour such as tabs taking a long time to suspend, or chrome crashing unexpectedly, I suggest you try turning this feature off.
